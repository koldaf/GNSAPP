Web Search Tools

The most reliable information sources on the internet are databases. A database is a collection of structured records organized in such a way to permit orderly retrieval, e.g for research, study, or administration. A database may contain citations or it may have full-text articles. Every database has its own subject scope and coverage. It is searchable in a variety of ways, such as tittle, author, subject, and keyword, but only in its normal database environment, using the database-specific search engine. There are Bibliographic databases, Full-text databases, Numeric databases, Image databases, Audio/Video databases, Mixed databases.

A database is designed in such a way that the user can retrieve a shortlist of relevant articles. The aim of database searching for reference purpose is (often) to retrieve a collection of say 50, mostly relevant documents, primarily journal articles and preferably in full text and in single query. Several search actions are normally needed to get that query or search strategy. Database search is a process. It involves the following activities:

  • Preparation: Identifying key concept, search terms
  • Decision on subject or keyword search (or both)
  • Constructing a query
  • Building a keyword search statement (start simple).
  • Limiting focus search results
  • Screening search results by some variable (language, publication date, etc.). The initial statement can be limited (pre-limit), or post-limit after reviewing results (recommended approach).
  • Refining to expand or reduce results

It is rare to get good results on the first try. If you get too many results, add new concepts or add limits to your research. If you get few results, remove concepts that are not critical, remove any limits. Some databases require subscription while others do not.

There is a comprehensive list of free and subscription online academic databases. http://www.wholeagain.com/free_academic_databases.html http://en.wikipedia.org/wiki/list_of_academic_databases_and_search_engines It is important to point out that subscription databases like The Essential Electronic Agriculture Library (TEEAL) and the E-granary databases can be used without Internet connection.
